# Coaching Health Tracker Prototype

This is a static prototype for a coach-facing performance and workload dashboard.

Open `index.html` in a browser to try it. The dashboard simulates live readings for 10 athletes, including individualized heart-rate zones, round load, recovery, readiness, alerts, a round timer, and coach notes.

The prototype is positioned as a performance platform, not a medical monitoring system. A production version should include athlete consent, secure storage, audit logs, and clear controls for stopping health-data sharing.
